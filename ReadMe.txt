How to run:

1. Load the folder db onto Eclipse IDE
2. You need to change the connection in the application.properties: "jdbc:mysql://localhost:3306/mydb", "user_name", "password"
    2.1 Create database in the server as given in another file "Database_Files" all the tables
    2.2 run the queries and change the path of the normalized csv files.
3. Open the Application.java in the main-method package and run the file on Tomcat server 7.0
4. GUI will  open and you can use the functionalities.
